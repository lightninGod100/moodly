Moodly App Summary
What is Moodly?
A mood tracking application where users can:

Track their daily moods with timestamps
View personal mood statistics and trends
See global mood statistics from all users
Get insights about their mood patterns


Current Architecture
Frontend (React/TypeScript/Vite/Tailwind)

Authentication Flow: Landing → Signup/Login → Dashboard
Main Components: Mood selector, user dashboard, global statistics
Navigation: Separate navbars for authenticated/public users
State Management: Local storage for mood history (temporary)

Backend (Node.js/Express/PostgreSQL)

Database: Users and moods tables with proper relationships
Authentication: JWT-based with bcrypt password hashing
API Structure: Separate routes for auth and moods
Middleware: Token authentication protection
